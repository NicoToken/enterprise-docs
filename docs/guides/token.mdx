# Create a Community token DAO

Follow the steps below to [create a new Community Token DAO](#create-a-new-community-token-dao) or [create a Community Token DAO using an existing token](#create-a-community-token-dao-using-an-existing-token). For more info on Community Token DAOs, visit the [concepts page](../concepts/about.mdx#community-token-daos).

## Prerequisites

Before you can create a Community Token DAO, make sure you have completed the following:

- Download Google Chrome
- Install [the Terra Station Chrome Extension](https://docs.terra.money/learn/terra-station/Download/terra-station-extension)
- At least two [Terra wallet addresses](https://docs.terra.money/learn/terra-station/Download/terra-station-extension#create-a-wallet)

## Connect your wallet to Enterprise

1. Sign in to your Terra Station wallet using the [Terra Station Browser Extension](https://setup-station.terra.money/). 

2. To connect your wallet to Enterprise, click the wallet icon in the upper right corner of the screen. 

3. Select **Terra Station Wallet**. 

4. If needed, click **approve** in the Terra Station extension. 

You are now connected to your wallet. Before you continue, make sure your [network is set to the **mainnet**](https://docs.terra.money/learn/terra-station/networks).

## Create a new Community Token DAO

1. Click on the plus sign on the left side of the screen and select **Community Token DAO**. 

2. Input a name for your DAO and upload a logo. Click **Next**.

3. Enter your token parameters, including decimal places, a token name, and a token symbol. Click **Next**.

4. Enter the wallet addresses you want to receive the initial distribution of tokens. Be sure to specify the amount each wallet will receive. Click **Next**.

5. Enter a token description, a token logo URL, the address of the marketing owner, and a project name. Click **Next**.

6. Define your [threshold](../concepts/config.mdx#threshold), [quorum](../concepts/config.mdx#quorum), [unlocking period](../concepts/config.mdx#unlocking-period), and [vote duration parameters](../concepts/config.mdx#vote-duration). Click **Next**.



<Admonition type="caution" title="Updating parameters">

After creation, [general parameters](../concepts/config.mdx#general-parameters) can only be updated using a [DAO configuration proposal](../concepts/governance.mdx#dao-configuration-proposals). 

</Admonition>

7. Enter any social media links you wish to add. Click **Next**.

8. Input the password for your Terra Station browser extension wallet and click **Post**.

Congratulations! You've just made a Community Token DAO!

## Create a Community Token DAO using an existing token

Follow the steps below to make a DAO using an existing [CW-20 token contract address](https://docs.cosmwasm.com/cw-plus/0.9.0/cw20/spec/#messages-1).

1. Click on the plus sign on the left side of the screen and select **Community Token DAO**. 

2. Input a name for your DAO and upload a logo. Click **Next**.

3. Select **Yes, find my existing token** and input your existing token contract address. Click **Next**. 

4. Define your [threshold](../concepts/config.mdx#threshold), [quorum](../concepts/config.mdx#quorum), [unlocking period](../concepts/config.mdx#unlocking-period), and [vote duration parameters](../concepts/config.mdx#vote-duration). Click **Next**.


<Admonition type="caution" title="Updating parameters">

After creation, [general parameters](../concepts/config.mdx#general-parameters) can only be updated using a [DAO configuration proposal](../concepts/governance.mdx#dao-configuration-proposals). 

</Admonition>

5. Enter any social media links you wish to add. Click **Next**.

6. Input the password for your Terra Station browser extension wallet and click **Post**.

Congratulations! You've just made a Community Token DAO!
